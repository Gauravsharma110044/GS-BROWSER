## Installation

1. Clone the repository to your local machine:

   ```bash
   git clone https://github.com/Chetan3327/ChatGPT.git
   ```
2. Navigate to the directory:

   ```bash
   cd ChatGPT
   ```
3. Install dependencies:

   ```bash
   npm install
   ```
4. run the development server:

   ```bash
   npm run dev
   ```
