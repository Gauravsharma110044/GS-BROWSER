# AI Notes

A universal AI note-taking and planning tool with auto-organization, action extraction, and reminders.

## Features

- Smart note-taking
- Auto-organization
- Action item extraction
- Predictive reminders

## Tech Stack

- Desktop: Electron or PyQt5
- Mobile: Flutter
- AI Backend: Python (Gemini/OpenAI)
